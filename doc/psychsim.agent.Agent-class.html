decide(self, vector, horizon=None, others=None, model=None, selection=None, actions=None, keys=None)

source code 

Generate an action choice for this agent in the given state

Parameters:
  • vector (KeyedVector) - the current state in which the agent is making its decision
  • horizon (int) - the value function horizon (default is use horizon specified in model)
  • others (str→ActionSet) - the optional action choices of other agents in the current time step
  • model (str) - the mental model to use (default is model specified in vector)
  • selection (str) - how to translate value function into action selection
    • random: choose one of the maximum-value actions at random
    • uniform: return a uniform distribution over the maximum-value actions
    • distribution: return a distribution (a la quantal response or softmax) using rationality of the given model
    • consistent: make a deterministic choice among the maximum-value actions (default setting for a model)
    • None: use the selection method specified by the given model (default)
  • actions - possible action choices (default is all legal actions)
  • keys - subset of state features to project over (default is all state features)

value(self, vector, action=None, horizon=None, others=None, model=None, keys=None)

source code 

Computes the expected value of a state vector (and optional action choice) to this agent

Parameters:
  • vector (KeyedVector) - the state vector (not distribution) representing the possible world under consideration
  • action (ActionSet) - prescribed action choice for the agent to evaluate; if None, then use agent's own action choice (default is None)
  • horizon (int) - the number of time steps to project into the future (default is agent's horizon)
  • others (str→ActionSet) - optional table of actions being performed by other agents in this time step (default is no other actions)
  • model - the model of this agent to use (default is True)
  • keys - subset of state features to project over in computing future value (default is all state features)

addModel(self, name, **kwargs)

source code 

Adds a new possible model for this agent (to be used as either true model or else as mental model another agent has of it). Possible arguments are:

  • R: the reward table for the agent under this model (default is True), KeyedTree→float
  • beliefs: the beliefs the agent has under this model (default is True), MatrixDistribution
  • horizon: the horizon of the value function under this model (default is True),int
  • level: the recursive depth of this model (default is True),int
  • rationality: the rationality parameter used in a quantal response function when modeling others (default is 10),float
  • discount: discount factor used in lookahead
  • selection: selection mechanism used in decide
  • parent: another model that this model inherits from (default is True)
Parameters:
  • name (sotr) - the label for this model
Returns: dict
the model created
